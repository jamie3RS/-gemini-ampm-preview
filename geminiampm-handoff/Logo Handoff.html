The four assets

Light surface → grey-wordmark file · dark surface → -rev (cream) file. The roundels are the client's exact artwork in every file — never recolour or rebuild them. Size by height, width:auto.

Where each goes

  • Header / nav: the notag pair. Over the dark hero/night sections use -notag-rev (cream); over a light/white header use -notag (grey).
  • Footer: dark background → gemini-logo-rev.png (full, with strapline — there's room).
  • Favicon / social / app icon: crop a roundel or the wordmark from notag, or request a square master.

Drop-in markup

Ship both colourways and let CSS pick by header state (the site is night-led, so cream is the common case). Full snippet + the single-image fallback are in the markdown.

Remove any .glogo / .glogo-word / .glogo-ampm CSS-circle lockup.

Add <img src="/assets/gemini-logo-notag-rev.png" alt="Gemini AMPM" height="30"> (or the light/dark swap pair).

Never set both width and height, stretch it, recolour the roundels, or re-typeset the wordmark.
